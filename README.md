# Form-automation-plugin

Chrome 表单自动化插件

在刷博客的时候，最麻烦的事情之一就是手动填写各种表单。为了提高我的冲浪体验，诞生了这款表单自动化插件，兼容绝大数博客，插件纯绿色，无隐私可言。

## 使用方法
clone到本地，浏览器打开：chrome://extensions/，加载已解压的扩展程序
